Before you start

Setup checklist

  • Log in as demo account — demo@octively.com
  • Have Noor-Dental-Clinic-Guide.md ready to upload (clip 4)
  • Incognito window · 100% zoom · maximised · notifications OFF
  • Game Bar set to 1080p / 30fps · mic OFF (see recorder tip at bottom)
  • Move the mouse slowly · pause ~2 s on every key screen
  • Start each clip ~1 s before the action, end ~1 s after
    • Two bots: Noor Dental for clips 1–6 (create, train, deploy). Auraline Cosmetics for clips 7–8 and the portal.
    Record clip 4 before clips 5 and 6. The KB must reach "Ready" before you demo the embed and live preview — the bot needs the dental guide loaded to answer questions.

    Recording on Windows — Xbox Game Bar (built-in, no install)

  • Press Win + G → Settings → set 1920×1080 / 30fps once.
  • Press Win + Alt + R to start recording. Same shortcut to stop.
  • Files save automatically to Videos\Captures\ as MP4.
  • Make sure the browser window is focused before pressing start.
  • Mic is OFF by default — leave it that way.
  • Rename each file immediately: admin-01-dashboard.mp4 etc.
